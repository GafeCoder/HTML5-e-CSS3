As imagens inseridas devem ser no formato .PMG ou .jpeg (JPG).
Para inserir a imagem coloque <img src="" alt="" >. informando os valores de diretorio da imagem.

Também é possível incluir uma imagem interna, utilazando a url da imagem já hospedade em um local, como valor de src=, o ponto negativo é que se o servidor terceiro cair, a declaração quebra.
